Notes

  • ul tag is used to define unordered lists
  • ol tag is used to define ordered list
  • li tag is used to add items to the list

Ordered Lists

  • type attribute is used to define the type of index for example a for abcd, i for roman numbers, etc
  • start attribute is used to set the initial index value
  • reversed attribute is used to reverse the order
