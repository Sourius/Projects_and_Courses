Notes

  • b tag makes text appear bold but strong tag is recommended
  • i tag makes text appear in cursive but em tag is recommended
  • u tag makes a text appear underlined but ins tag is recommended
  • sup tag makes text appear high up in the air (supper script)
  • sub tag makes text appear low down in the air (sub script)
  • del tag makes taxt appear cross down, same as strike tag in html 4
  • hr tag adds an horitzontal line
