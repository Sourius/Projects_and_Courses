Notes

  • form tag is used to declare a form
  • input is a basic element of the form, it is used to read data
  • type attribute specifies the type of input: text, checkbox, radio
  • select tag is used to specify a dropdown
  • option tag is used to declare an option inside a select tag
  • input type submit is used to declare a submit button

Input

  • value attribute is used to define the default value or the text of the input
  • placeholder attribute is used to give hint

Radio buttons

  • name attribute is used to link different radio buttons as a same type
  • value attribute is used to specify which radio button was clicked

Checkbox

  • checked attribute specifies de default value

Select

  • selected attribute specifies the default option value
