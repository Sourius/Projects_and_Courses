Notes

  • to access html elements from css we use selectors
  • span tag is used to to group or make a chunk of text or paragraphs, to make it easier to apply css
  • div tag is used to group html elements inside a block

Class

  • we can associate a html element with class, with the property class
  • the selector for the class is .
  • we can add more than one class to html element, seperated by space

Id

  • we can asociate an html element with an id, with the property id
  • the selector for an id is #
  • each html element can have only one id and that must be unique
